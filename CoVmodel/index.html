Piecewise SIR models for COVID-19 breakout

This repository contains

  • a Jupyter notebook that uses `julia` and `QuadDIRECT` to perform discrete simulation and optimisation to fit a simple piecewise SIR model to John Hopkins case count data. The piecewise nature of the model allows the user to observe the effect of government interventions.
  • a javascript code for running discrete piecewise SIR simulations in the web browser. Data from John Hopkins, cleaned by the julia notebook above is loaded by the script and then the user can experiment with the SIR parameters and obtain simulated predictions of how the pandemic will play out.
