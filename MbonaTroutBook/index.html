Trout Hatchery Manual, Mbona Private Nature Reserve
by Hugh Murrell, Pierre Olivier and Dennis Dyer

Latex Source on GitHub

Greetings Trout enthusiasts!

This manual was created to record proceedures for growing Rainbow Trout from eyed-eggs up until 12 inch fish ready for release into dams. This document is placed on github under Creative Commons Licence to aid other Trout enthusiasts out there. Our Hatchery is at Mbona Private Nature Reserve in KwaZulu-Natal, see www.mbona.co.za for further details.
